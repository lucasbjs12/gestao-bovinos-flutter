Política de Privacidade

Aplicativo Gestão de Rebanho

Última atualização: 03 de junho de 2026

1. Quem somos

O aplicativo Gestão de Rebanho é um app de gestão de propriedades rurais e rebanhos bovinos, desenvolvido e mantido por Lucas Borges Ezirio Ribeiro ("nós"). Esta política explica quais dados coletamos, como os usamos e quais são os seus direitos.

2. Dados que coletamos

Coletamos apenas os dados necessários para o funcionamento do app:

  • Dados de conta: e-mail e nome da fazenda, usados para autenticação e identificação da sua conta.
  • Dados do rebanho: informações que você cadastra — bovinos, invernadas, manejos sanitários, movimentações e baixas.
  • Fotos: imagens dos animais que você opcionalmente envia.

Não coletamos localização, contatos, nem dados de outros aplicativos. Não usamos seus dados para publicidade.

3. Como usamos seus dados

  • Para autenticar o seu acesso e manter sua conta.
  • Para armazenar e sincronizar os dados da sua fazenda entre seus aparelhos.
  • Para enviar e-mails essenciais (confirmação de conta e recuperação de senha).

4. Onde os dados são armazenados

Utilizamos provedores de infraestrutura confiáveis:

  • Google Firebase (autenticação e banco de dados na nuvem) — dados armazenados em servidores do Google.
  • Cloudinary (armazenamento das fotos dos animais).

Cada conta tem seus dados isolados: nenhuma fazenda acessa os dados de outra.

5. Compartilhamento

Não vendemos nem compartilhamos seus dados com terceiros para fins comerciais. Os dados são processados apenas pelos provedores de infraestrutura citados acima, estritamente para operar o aplicativo.

6. Seus direitos (LGPD)

Você pode, a qualquer momento:

  • Acessar e corrigir os dados da sua conta dentro do app.
  • Excluir sua conta e todos os dados em Perfil → Excluir minha conta. A exclusão é permanente e remove seus dados da nuvem.
  • Solicitar informações sobre seus dados pelo e-mail de contato abaixo.

7. Exclusão de conta e dados

Para excluir sua conta e todos os dados associados, acesse Perfil → Excluir minha conta dentro do aplicativo. Caso não tenha mais acesso ao app, envie um pedido para lucasbjs12@hotmail.com que providenciaremos a exclusão.

8. Segurança

Adotamos medidas para proteger seus dados, incluindo autenticação por senha, conexões criptografadas e regras de acesso que isolam os dados de cada conta. Nenhum sistema é 100% infalível, mas trabalhamos para manter suas informações seguras.

9. Retenção

Mantemos seus dados enquanto sua conta existir. Ao excluir a conta, os dados são removidos dos nossos sistemas de nuvem.

10. Menores de idade

O aplicativo é destinado a produtores rurais maiores de 18 anos e não é direcionado a menores de idade.

11. Alterações nesta política

Podemos atualizar esta política periodicamente. A data da última atualização aparece no topo desta página.
